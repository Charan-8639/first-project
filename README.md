🔐 Password Generator
📌 Description

This is a simple Python-based Password Generator that creates strong and secure random passwords using letters, numbers, and special characters. It helps users generate safe passwords instantly.

🚀 Features
Generates strong random passwords
User can choose password length
Includes uppercase, lowercase, numbers, and symbols
Fast and easy to use
🛠️ Technologies Used
Python 🐍
Random module
String module
▶️ How to Run
Make sure Python is installed on your system
Download or clone this repository
Open terminal or command prompt
Run the file using:
python password_generator.py
💻 Example Output
🔐 Password Generator
Enter password length: 10
Your secure password is: A@9kLp#2Xz
🎯 Purpose

This project was created to practice Python programming and understand how random functions and strings work in real applications.
